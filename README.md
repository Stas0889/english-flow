# English Flow

English Flow — рабочий MVP веб-приложения для изучения английского языка уровня A1 для русскоязычного пользователя.

Стек:

- React
- Vite
- React Router (`HashRouter`)
- localStorage
- Web Speech API
- обычный CSS

## Что умеет MVP

- 5 новых слов в день
- интервальное повторение по стадиям SRS
- карточки новых слов и повторения
- словарь с фильтрами
- раздел ошибок
- базовая грамматика A1
- статистика прогресса
- экспорт и импорт данных из localStorage
- готовая сборка для GitHub Pages

## Установка

```bash
npm install
```

## Локальный запуск

```bash
npm run dev
```

Открой адрес, который покажет Vite в терминале, обычно `http://localhost:5173/`.

## Сборка

```bash
npm run build
```

Готовая статическая сборка будет в папке `dist/`.

## Предпросмотр production-сборки

```bash
npm run preview
```

## Публикация на GitHub Pages

Приложение уже настроено под GitHub Pages:

- используется `HashRouter`, поэтому маршруты не ломаются;
- в `vite.config.js` задан относительный `base: './'`;
- добавлен workflow `.github/workflows/deploy.yml`.

Чтобы опубликовать:

1. Создай GitHub-репозиторий и отправь туда содержимое проекта в корень репозитория.
2. В настройках репозитория открой `Settings -> Pages`.
3. В `Build and deployment` выбери `GitHub Actions`.
4. Запушь изменения в ветку `main`.
5. GitHub Actions соберёт `dist` и опубликует приложение.

Если workflow не нужен, можно вручную залить содержимое `dist/` на любой статический хостинг.
